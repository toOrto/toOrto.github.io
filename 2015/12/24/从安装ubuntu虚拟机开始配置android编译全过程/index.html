ubuntu_14.04_android-6.0.1_r3

下载ubuntu:64位、Ubuntu_14.04

设置root密码

安装虚拟机，之后设置root密码。

  • sudo passwd
  • 输入密码
  • 确认密码

安装VMtools:

  • mkdir VMwaretools
  • cp -R /media/**/VMware Tools/ ./
  • cd VMware Tools
  • sudo tar zxvf VMwareTools-10.0.0-2977863.tar.gz
  • cd vmware-tools-distrib/
  • sudo ./vmware-install.real.pl -d

安装Android Studio

  • sudo add-apt-repository ppa:paolorotolo/android-studio
  • sudo apt-get update
  • sudo apt-get install android-studio

中途会遇到
Network unreachable
解决办法 设置HTTP代理

如何为终端设置HTTP代理

export http_proxy=http://yourproxyaddress:proxyport

如果使用sudo执行apt-get
则在su root以后，再

export http_proxy=http://yourproxyaddress:proxyport

或者

Defaults env_keep = "http_proxy https_proxy ftp_proxy"

参考链接
若安装完，android studio不能启动程序，则重启即可。

开启ssh服务

sudo apt-get install openssh-server

/etc/init.d/ssh start

Android源码编译清理

  1. 在源码目录的根目录下，makeclean;
  2. 进到源码的\linux\kernel\目录下，执行makemrproper；
  3. 再退回到根目录，执行sourcebuild/envsetup.sh,lunch,make.
