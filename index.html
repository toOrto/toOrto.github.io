modify the APK

By toOrto

Published 2015-04-14

修改APK文件遇到的一些问题

问题1：
adb install xxx.apk时，出现：

Zip EOCD: expected >= 22 bytes, found 21

解决方法：

该问题是由于End of central directory record (EOCD)总长度不对造成的，笔者是由于少了一位0x00造成的。EOCD总长度本应该是22位，缺只有21位，则会报错。修改方法，检查缺失字段，并补齐。
